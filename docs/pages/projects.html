Mystery Word Game

I chose this game because of my love for word puzzles and I thought this would be a game I could enjoy. I like that the game is simple and basic word game and you can guess what the word is, it's simple but addicting. The words are positioned using Reg Exp and allow the users' selected letter to be revealed one by one the word without giving away the whole word at the same time.

Language Used:

Javascript ES6 | Node | CSS3 | HTML5
